# Flight Booking System - Full Stack Application

A complete flight booking system with dynamic pricing, wallet management, and PDF ticket generation.

## Features

- **Flight Search & Booking**: Browse available flights and book with dynamic pricing
- **Dynamic Pricing**: Prices increase by 10% for repeat bookings within 5 minutes, reset after 10 minutes

- **PDF Ticket Generation**: Automatic ticket creation with passenger details, PNR, and pricing
- **Booking History**: View all past bookings and download tickets anytime
- **RESTful API**: Full backend API for all operations
- **Database Integration**: MongoDB for persistent data storage

## Tech Stack

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB
- **PDF Generation**: PDFKit or similar library

### Frontend
- **Framework**: React
- **State Management**: React Hooks/Context
- **Styling**: CSS/Tailwind CSS
